post /patients

Register a new Patient user.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • name: required (string)
  • password: required (string)
  • therapist: (number)

Examples:

noTherapist:

{
  "name": "Greg",
  "password": "jfido"
}

withTherapist:

{
  "name": "Fred",
  "password": "2fei333",
  "therapist": 39012
}

HTTP status code 201

Succesfully registered.

Body

Media type: application/json

Type: object

Properties
  • name: required (string)
  • id: required (number)
  • therapist: required (object)
    • name: required (string)
    • id: required (number)

    Example:

    {
  "name": "UserA",
  "id": 8971203
}

Example:

{
  "name": "Joey",
  "therapist": {
    "name": "Jen",
    "id": 92013
  },
  "id": 128939
}

HTTP status code 403

Invalid username/password.

Body

Media type: application/json

Type: object

Properties
  • error: required (string)

Examples:

usernameTaken:

{
  "error": "Username taken already."
}

badPassword:

{
  "error": "Password must be at least 8 characters long."
}
